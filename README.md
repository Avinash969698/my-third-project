# my-third-project
Project Title: Big Basket Data Cleaning and Visualization

Description:
Performed data cleaning and preprocessing on Big Basket dataset to handle missing values, duplicates, and outliers using Python (Pandas, NumPy).
After cleaning, created interactive visualizations using Matplotlib and Seaborn to analyze product categories, pricing trends, and customer preferences.
The project helped uncover insights into top-selling products and pricing distribution across categories.
